## 🚀 3단계 빠른 시작

### 1️⃣ 패키지 설치 (5분)
```bash
npm install
```

### 2️⃣ 디렉토리 & 파일 정렬 (5분)
```bash
node setup.js
```
그런 후 `SETUP_INSTRUCTIONS.md`의 "파일 이동" 섹션을 따르세요.

### 3️⃣ 개발 서버 실행 (1분)
```bash
npm run dev
```

브라우저에서 **http://localhost:3000** 열기 ✨

## ✨ 프로젝트 기능

### 📰 뉴스레터 보기 (누구나)
- 메인 페이지에서 발행된 모든 뉴스레터 목록
- 각 카드: 제목, 작성자, 날짜, 요약
- 카드 클릭으로 상세 페이지 이동

### ✍️ 뉴스레터 작성 (비밀번호 필요)
- "새 글 쓰기" 클릭
- 비밀번호 입력 (`linkus_2026`)
- 제목, 작성자, 본문(마크다운) 입력
- 발행 완료

### 🗑️ 뉴스레터 삭제 (비밀번호 필요)
- 상세 페이지에서 "삭제" 버튼
- 비밀번호 입력
- 재확인 후 삭제

## 🔐 보안 특징

✅ **비밀번호 보호**
- 작성: 비밀번호 필수
- 삭제: 비밀번호 + 재확인 필수

✅ **이중 검증**
- 클라이언트 측: 모달에서 먼저 확인
- 서버 측: API에서 재확인

✅ **환경변수 관리**
- `.env.local`에만 저장
- 코드에 하드코딩 안 함
- 클라이언트 번들에 포함 안 함
