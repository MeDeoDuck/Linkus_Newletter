===============================================================================
TROUBLESHOOTING
===============================================================================

PROBLEM: "python is not recognized" or "node is not recognized"
SOLUTION: 
  - Make sure Python/Node is installed
  - Restart Command Prompt after installation
  - Use full path: C:\Python\python.exe DO_REORGANIZE.py

PROBLEM: "Access Denied" when copying files
SOLUTION:
  - Close the files in any text editor or IDE
  - Run Command Prompt as Administrator
  - Try batch script instead

PROBLEM: Script created directories but didn't copy files
SOLUTION:
  - Check that source files exist: dir lib.db.ts
  - Check that target directories were created: dir app\lib
  - Try manual copy commands from troubleshooting section
  - Check file permissions on source files

PROBLEM: Build fails after reorganization
SOLUTION:
  - Check that import paths use @/ alias: @/lib/db, @/components/PasswordModal
  - Verify tsconfig.json has correct path mappings
  - Check for any "file not found" errors in build output
