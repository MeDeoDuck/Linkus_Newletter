# Linkus Newsletter - 정적 블로그\n\n팀 내부용 정적 마크다운 블로그입니다.\n\n## 🚀 특징\n\n- ✅ 순수 정적 HTML (GitHub Pages 배포)\n- ✅ Markdown 지원\n- ✅ 자동 빌드 및 배포 (GitHub Actions)\n- ✅ 읽기 전용\n\n## 📝 포스트 추가하기\n\n1. `public/posts/` 디렉토리에 `.md` 파일 생성\n2. 다음 형식으로 작성:\n\n```markdown\n---\ntitle: 포스트 제목\nauthor: 작성자 이름\ndate: 2026-03-25\n---\n\n# 본문 내용\n\n마크다운으로 자유롭게 작성합니다.\n```\n\n3. Git에 커밋하고 푸시\n4. GitHub Actions가 자동으로 빌드하고 배포합니다!\n\n## 📚 Frontmatter 포맷\n\n| 필드 | 설명 | 예시 |\n|------|------|------|\n| `title` | 포스트 제목 | \"첫 번째 포스트\" |\n| `author` | 작성자 이름 | \"Admin\" |\n| `date` | 발행 날짜 (YYYY-MM-DD) | \"2026-03-25\" |\n\n## 🌐 배포\n\n- 사이트: https://MeDeoDuck.github.io/Linkus_Newletter\n- GitHub Actions 자동 배포\n\n## 🛠 개발\n\n```bash\n# 로컬 개발 서버\nnpm run dev\n\n# 정적 빌드\nnpm run build\n```\n\n## 📁 폴더 구조\n\n```\n.\n├── app/\n│   ├── page.tsx              # 블로그 목록\n│   ├── posts/\n│   │   └── [id]/\n│   │       └── page.tsx      # 포스트 상세\n│   ├── lib/\n│   │   └── posts.ts          # MD 파일 파싱\n│   └── layout.tsx\n├── public/\n│   └── posts/                # ← 포스트 MD 파일 저장 위치\n│       └── welcome.md\n└── ...\n```\n
